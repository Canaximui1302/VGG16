The zipped dataset for training can be downloaded here:
> https://storage.googleapis.com/download.tensorflow.org/example_images/flower_photos.tgz

To train the model:
* For Linux & MacOS:
> python3 VGG16.py

* For Windows PowerShell:
> python VGG16.py

To use the trained model:
* For Linux & MacOS:
> python3 test.py

* For Windows PowerShell:
> python test.py
